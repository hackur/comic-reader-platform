sample.cbz

Auto-generated demo comic for the open-source comic reader.
Created by tools/scripts/generate-sample-cbz.mjs.

  - ComicInfo.xml with Series=Sample, Number=1, PageCount=8,

and ZIP writer using only Node 22+ built-ins (node:zlib for deflate).

Regenerate with:
